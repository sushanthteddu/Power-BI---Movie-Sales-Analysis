**Movie Data Analysis Dashboard**

**📌 Project Overview**
This project analyzes a movie dataset to uncover trends in movie releases, genres, ratings, and box office revenue. The goal is to transform raw data into meaningful insights through an interactive dashboard.

**🎯 Objective**
Analyze movie trends over time

Identify top-performing movies, genres, and directors

Explore relationships between revenue, ratings, votes, and runtime

Build an interactive and insightful dashboard

**📊 Dataset Description**
Column	Description

Rank	Movie ranking position

Title	Name of the movie

Genre	Movie category (Action, Drama, etc.)

Description	Short summary of the movie

Director	Director name

Actors	Main cast

Year	Release year

Runtime (Minutes)	Duration of movie

Rating	IMDb rating

Votes	Number of votes

Revenue (Millions)	Box office revenue

Metascore	Critic score (0–100)

**📈 Key Analysis**

1️⃣ Movie Overview & Trends

Movies released per year

Revenue trends over time

Average rating trend

Runtime analysis

Peak revenue years

**2️⃣ Genre & Rating Analysis**

Genre distribution

Average ratings by genre

Revenue by genre

Metascore analysis

**3️⃣ Top Performers**

Top directors by revenue

Most voted movies

Highest-grossing movies

Key KPIs (Revenue, Votes, Movies)

**4️⃣ Correlation Analysis**

Revenue vs Rating

Revenue vs Votes

Runtime vs Revenue

Genre-wise yearly performance

**📊 Dashboard Features**

KPI Cards (Total Revenue, Avg Rating, Total Movies, Votes)

Bar & Column Charts


Line Charts (Yearly Trends)

Scatter Plots (Correlation insights)

Pie/Donut Charts (Genre split)

Slicers for filtering

**💡 Key Insights**

Certain genres consistently generate higher revenue

Movies with higher votes tend to have better ratings

Runtime has a moderate impact on revenue

Revenue trends show growth over the years
